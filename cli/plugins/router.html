# router

# 简介

router 插件为@xiyun/cli的generator命令提供了一个自动创建路由的功能。

# 使用方法

虽然 router 插件已在内部集成，可以直接在命令行中执行， 但还是需要放到 generator 插件中去调用。

内部调用方法：

import { router } from '@xiyun/xy-plugin-router';

// 执行后，就会自动创建好 foo 相关的路由文件
router('foo');
